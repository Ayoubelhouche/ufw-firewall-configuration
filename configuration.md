# Configuration Guide

Learn how to configure UFW for your needs.

## Basic Configuration
- Enable UFW:
  ```bash
  sudo ufw enable
  ```

- Allow SSH Connections:
  ```bash
  sudo ufw allow ssh
  ```

- Allow HTTP and HTTPS:
  ```bash
  sudo ufw allow http
  sudo ufw allow https
  ```
